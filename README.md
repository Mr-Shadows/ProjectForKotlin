# 🚚 DAP Доставка
DeliveryApp – это мобильное приложение для курьеров, предназначенное для упрощения процесса доставки заказов. Курьеры могут принимать и отслеживать заказы, управлять своей статистикой, получать выплаты и связываться с поддержкой через интуитивно понятный интерфейс.

# 📱 Функции приложения:

Список доступных заказов: Курьеры могут видеть заказы с фильтрацией по времени, расстоянию и оплате.

Навигация: Встроенная интеграция с картами для прокладывания маршрутов.

Статистика: Полная информация о доставках, рейтинге и заработке за день/неделю/месяц.

Баланс и выплаты: Курьеры могут отслеживать свой доход в реальном времени и выбирать предпочтительные методы выплат.

Поддержка в один клик: Быстрый доступ к техподдержке для решения проблем.

Уведомления: Настраиваемые уведомления для новых заказов и завершенных доставок.
